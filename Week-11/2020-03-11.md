## embed video in web page
```
playsinline: true, autoplay: true, loop: true, muted: true
```
- `loop` : 반복재생
- `autoplay` : 페이지 로딩되면 자동재생
- `muted` : 음소거, `autoplay` 하기 위해 같이 사용되어야 함
- `playsinline` : Safari에서 재생되려면 필요함
